# Cheating

**Warning: if you use a cheat you will not be able to go to the next level even if you cancel the cheat**

Debugging cheats are attached to the following buttons:

- `F9` - Immortality mode;
- `F10` - Takes all guns;
- `F11` - Endless bullets mode;
- `F12` - Ghost mode;
